3 Pendekatan Modernisasi

Stripe-style light mode — List + Form + Dashboard

A

Surface Polish — CSS Only

Override semua 3 area sekaligus, pure CSS. Cepat, aman, konsisten dengan cara kerja saat ini. Risiko: komponen Frappe tertentu susah di-style tanpa JS (empty state, filter chips).

Pro

  • 1 PR, cepat ship
  • Zero JS risk

Con

  • Beberapa area tidak bisa 100%
B

Deep Polish — CSS + JS

CSS untuk layout/warna, JS untuk elemen dinamis: empty state custom, filter chips aktif, sticky action bar di form. Hasil paling lengkap.

Pro

  • Tampilan paling solid
  • Cover semua edge case

Con

  • Lebih banyak JS → risiko konflik Frappe
C ★

Phased — Satu Area Per PR (Rekomendasi)

List view dulu → validasi → Form view → validasi → Dashboard. Setiap fase shippable sendiri. CSS + JS sesuai kebutuhan tiap area.

Pro

  • Validasi bertahap
  • Rollback mudah per area
  • Risk kecil

Con

  • 3 PR, lebih lama total
